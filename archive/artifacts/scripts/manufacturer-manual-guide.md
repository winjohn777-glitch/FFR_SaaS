# Real Manufacturer Installation Manual Guide

This guide provides the exact locations and steps to obtain actual manufacturer installation manuals for each 1500 series SOP roofing system.

## How to Obtain Real Installation Manuals

### Method 1: Direct Manufacturer Access
Most manufacturers require registration on their professional contractor portals to access installation manuals.

### Method 2: Authorized Dealer Networks
Contact your local authorized dealers who have access to complete installation libraries.

### Method 3: Trade Account Access
Establish trade accounts with manufacturers for direct access to technical resources.

## Automated Collection Strategy

### For Florida First Roofing Implementation:

1. **Establish Trade Accounts** with all manufacturers listed above
2. **Assign team member** to maintain manufacturer portal access
3. **Schedule quarterly updates** to download latest installation guides
4. **Organize by SOP number** in the DMS manufacturer-guides folder
5. **Create internal naming convention** for consistent file management

### File Naming Convention:
```
[SOP-CODE]-[MANUFACTURER]-[PRODUCT]-Installation-Guide.pdf

Examples:
GAF-TIM-HVHZ-GAF-Timberline-HDZ-Installation-Guide.pdf
CAR-SUR-HVHZ-Carlisle-Sure-Weld-TPO-Installation-Guide.pdf
```

### DMS Integration Points:
- Upload to Documents → Manufacturer Guides folder
- Tag with: manufacturer name, product type, SOP number, HVHZ status
- Link to corresponding 1500 series SOP
- Set review dates for manual updates
