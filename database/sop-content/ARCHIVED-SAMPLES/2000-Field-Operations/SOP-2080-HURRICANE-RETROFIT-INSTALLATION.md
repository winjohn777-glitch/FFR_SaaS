# SOP-2080: Hurricane Retrofit Installation Procedures

## 1. PURPOSE AND SCOPE

### 1.1 Purpose
This Standard Operating Procedure (SOP) establishes comprehensive protocols for hurricane retrofit installations to strengthen existing roofing systems against extreme weather conditions. This procedure ensures compliance with Florida Building Code High Velocity Hurricane Zone (HVHZ) requirements and provides enhanced storm resistance for existing structures.

### 1.2 Scope
This SOP applies to all Florida First Roofing LLC hurricane retrofit projects including:
- Roof-to-wall connection strengthening
- Enhanced sheathing attachment systems
- Secondary water barriers and protection
- Impact-resistant material upgrades
- Ventilation system storm protection
- Gutter and drainage system reinforcement

## 2. DEFINITIONS

- **Hurricane Retrofit**: Modification of existing structures to improve resistance to hurricane-force winds
- **HVHZ (High Velocity Hurricane Zone)**: Areas subject to hurricane winds of 120+ mph (Miami-Dade and Broward counties)
- **Uplift Resistance**: Ability of roofing system to resist wind-induced upward forces
- **Secondary Water Barrier**: Additional protection against water intrusion during storm events
- **Hurricane Straps**: Metal connectors that strengthen roof-to-wall connections
- **Storm Shutters**: Protective coverings for windows and openings during storms

## 5. PROCEDURES

### 5.1 Pre-Installation Assessment

#### 5.1.1 Structural Evaluation
1. **Existing Roof System Analysis**
   - Roof framing member size and spacing evaluation
   - Sheathing type, thickness, and attachment assessment
   - Existing fastener pattern and condition review
   - Load path analysis from roof to foundation

2. **Wind Load Calculation**
   - Determine design wind speeds for location
   - Calculate uplift forces for roof components
   - Assess existing vs. required capacity
   - Identify deficient areas requiring reinforcement

3. **Building Envelope Assessment**
   - Window and door opening vulnerability evaluation
   - Wall-to-roof connection adequacy review
   - Existing secondary water barrier assessment
   - Ventilation system storm readiness evaluation

#### 5.1.2 Code Compliance Review
1. **Current Code Requirements**
   - Review applicable building code editions
   - Identify grandfather clause applicability
   - Determine required vs. voluntary upgrades
   - Assess permit and inspection requirements

2. **Engineering Requirements**
   - Determine if structural engineer review is required
   - Coordinate structural calculations if needed
   - Verify load path continuity
   - Confirm connection capacity requirements

### 5.2 Roof-to-Wall Connection Strengthening

#### 5.2.1 Hurricane Strap Installation
1. **Material Selection**
   - Use galvanized or stainless steel straps
   - Verify strap capacity ratings for application
   - Ensure compatibility with existing framing
   - Follow manufacturer specifications for fasteners

2. **Installation Procedures**
   - Locate and mark strap attachment points
   - Pre-drill holes to prevent wood splitting
   - Install straps with appropriate fasteners
   - Verify minimum edge distances and spacing

3. **Quality Control**
   - Inspect each strap for proper installation
   - Verify fastener type and quantity
   - Check for proper wood contact
   - Document installation with photographs

#### 5.2.2 Continuous Load Path Creation
1. **Truss-to-Wall Connections**
   - Install additional clips at each truss bearing point
   - Verify connection to top plate and wall studs
   - Ensure load transfer to foundation
   - Use approved connector hardware

2. **Rafter-to-Wall Connections**
   - Add hurricane ties at each rafter location
   - Install blocking between rafters as needed
   - Verify connections to wall plate system
   - Check for proper fastener penetration

### 5.3 Enhanced Sheathing Attachment

#### 5.3.1 Fastener Upgrade Procedures
1. **Existing Fastener Assessment**
   - Identify current fastener pattern and spacing
   - Determine fastener type and condition
   - Calculate existing vs. required capacity
   - Plan additional fastener locations

2. **Additional Fastener Installation**
   - Use ring-shank nails or screws for enhanced holding power
   - Install fasteners at 4" on center on panel edges
   - Install fasteners at 6" on center in field of panel
   - Avoid over-driving fasteners that could reduce capacity

3. **Panel Edge Treatment**
   - Ensure proper support at all panel edges
   - Install blocking where panel edges are unsupported
   - Use H-clips between panels where required
   - Check for proper panel gap spacing

#### 5.3.2 Sheathing Replacement When Required
1. **Removal Procedures**
   - Carefully remove existing inadequate sheathing
   - Preserve structural framing during removal
   - Identify and address any framing damage
   - Clean surface for new sheathing installation

2. **New Sheathing Installation**
   - Use minimum 15/32" plywood or 7/16" OSB
   - Install with proper fastener pattern
   - Maintain required spacing between panels
   - Ensure proper orientation for maximum strength

### 5.4 Secondary Water Barrier Systems

#### 5.4.1 Self-Adhering Membrane Installation
1. **Surface Preparation**
   - Clean sheathing surface of debris and oils
   - Repair any sheathing damage or irregularities
   - Prime surface if required by manufacturer
   - Ensure surface is dry before application

2. **Membrane Application**
   - Start at lowest point and work upward
   - Overlap seams minimum 3 inches
   - Press firmly to ensure proper adhesion
   - Seal around penetrations with compatible materials

3. **Quality Assurance**
   - Inspect for proper adhesion and seams
   - Check for wrinkles or air bubbles
   - Verify complete coverage of required areas
   - Test seams for water resistance

#### 5.4.2 Enhanced Flashing Systems
1. **Critical Detail Areas**
   - Install enhanced flashing at all roof penetrations
   - Upgrade valley and transition area flashing
   - Install continuous ridge and hip flashing
   - Enhance gutter and drainage system connections

2. **Installation Techniques**
   - Use corrosion-resistant flashing materials
   - Ensure proper overlap and fastening
   - Seal all fastener penetrations
   - Test all flashing details for water resistance

### 5.5 Impact-Resistant Material Upgrades

#### 5.5.1 Impact-Resistant Shingles
1. **Material Selection**
   - Choose Class 4 impact-resistant shingles
   - Verify manufacturer wind speed ratings
   - Ensure compatibility with existing system
   - Check insurance qualification requirements

2. **Installation Modifications**
   - Use 6 nails per shingle instead of 4
   - Install nails in manufacturer-designated locations
   - Use corrosion-resistant nails
   - Follow enhanced installation procedures

#### 5.5.2 Metal Roofing Upgrades
1. **Standing Seam Systems**
   - Use continuous structural seaming
   - Install with enhanced clip spacing
   - Verify thermal movement accommodation
   - Ensure proper fastener placement

2. **Screw-Down Systems**
   - Use self-sealing screws with EPDM washers
   - Install fasteners in flat areas only
   - Follow manufacturer's fastener pattern
   - Apply sealant at lap seams

### 5.6 Ventilation System Storm Protection

#### 5.6.1 Ridge Vent Modifications
1. **Wind-Resistant Ridge Vents**
   - Install ridge vents rated for high winds
   - Use continuous fastening systems
   - Seal end caps and joints
   - Install wind deflectors if required

2. **Storm Closure Systems**
   - Install removable storm covers for extreme events
   - Design closure system for easy operation
   - Ensure proper ventilation when covers removed
   - Test closure system operation

#### 5.6.2 Soffit and Eave Ventilation
1. **Protected Intake Vents**
   - Install vents with integral rain guards
   - Use corrosion-resistant materials
   - Ensure proper screening against debris
   - Maintain required ventilation rates

2. **Emergency Closure Capabilities**
   - Design temporary closure systems
   - Provide easy access for storm preparation
   - Ensure quick installation/removal
   - Document closure procedures
