# SOP-2075: Structural Repair and Reinforcement Procedures

## 1. PURPOSE AND SCOPE

### Purpose
To establish comprehensive procedures for structural repair and reinforcement activities related to roofing projects for Florida First Roofing LLC, ensuring structural integrity, code compliance, and safety while addressing Florida's unique challenges including hurricane damage, high winds, aging structures, and foundation settlement issues.

### Scope
This SOP applies to all structural repair and reinforcement activities including:
- Roof deck repair and replacement
- Rafter and truss reinforcement
- Foundation and support structure repair
- Hurricane damage restoration
- Structural upgrades for code compliance
- Seismic retrofitting (limited applications)
- Load capacity enhancement
- Weather-related damage repair

## 2. DEFINITIONS

**Structural Repair**: The restoration of a structural component to its original load-carrying capacity and function.

**Structural Reinforcement**: The strengthening of existing structural elements to carry additional loads or meet current building codes.

**Load Path**: The route through which loads are transferred from their point of application to the foundation.

**Structural Engineering**: The professional analysis and design of structural systems by a licensed engineer.

**Engineered Solution**: A structural repair or reinforcement designed by a licensed structural engineer.

**Code Compliance**: Meeting current building code requirements for structural performance.

**Hurricane Strapping**: Metal connectors designed to resist uplift forces from high winds.

## 7. REPAIR PROCEDURES

### 7.1 Roof Deck Repair
**Plywood/OSB Deck Repair:**
1. **Damage Assessment**
   - Identify damaged areas
   - Check supporting structure
   - Verify remaining capacity
   - Plan removal sequence

2. **Removal Process**
   - Remove roofing materials
   - Cut out damaged deck areas
   - Preserve structural framing
   - Clean preparation areas

3. **Installation Process**
   - Install new deck materials
   - Match existing thickness
   - Secure with proper fasteners
   - Integrate with existing deck

**Structural Deck Requirements:**
- Use structural grade materials
- Meet span requirements
- Provide proper edge support
- Install hurricane strapping

### 7.2 Rafter and Truss Repair
**Rafter Reinforcement:**
1. **Sister Rafter Installation**
   - Install full-length sister rafters
   - Use structural grade lumber
   - Connect with approved fasteners
   - Maintain proper spacing

2. **Splice Repair**
   - Design engineered splices
   - Use steel plates or lumber
   - Provide adequate connection
   - Transfer loads properly

**Truss Repair:**
- Use engineered repair procedures
- Maintain truss geometry
- Replace damaged members
- Reinforce connections

### 7.3 Foundation Repair
**Concrete Foundation Repair:**
1. **Crack Repair**
   - Clean crack thoroughly
   - Inject epoxy or polyurethane
   - Apply surface treatments
   - Monitor for recurrence

2. **Section Replacement**
   - Remove damaged concrete
   - Prepare reinforcement
   - Place new concrete
   - Cure properly

**Masonry Foundation Repair:**
- Repoint deteriorated mortar
- Replace damaged blocks
- Install moisture barriers
- Provide proper drainage

### 7.4 Hurricane Retrofitting
**Roof-to-Wall Connections:**
1. **Hurricane Straps Installation**
   - Install on every rafter/truss
   - Use approved connector types
   - Follow manufacturer specifications
   - Provide continuous load path

2. **Enhanced Connections**
   - Upgrade existing connections
   - Add supplemental fasteners
   - Install tie-down systems
   - Strengthen wall connections

**Opening Protection:**
- Install impact-resistant windows
- Provide shutters or panels
- Reinforce door assemblies
- Upgrade garage doors

### 7.5 Steel Reinforcement
**Structural Steel Additions:**
1. **Beam Installation**
   - Size beams properly
   - Provide adequate connections
   - Transfer loads correctly
   - Coordinate with other trades

2. **Column Installation**
   - Install on proper footings
   - Size for required loads
   - Connect to beam systems
   - Provide lateral stability

**Connection Details:**
- Use engineered connections
- Provide proper welds
- Install galvanized hardware
- Detail for corrosion protection
