# SOP-1041: Safety Data Sheet (SDS) Management

## 1. PURPOSE AND SCOPE

### 1.1 Purpose
This Standard Operating Procedure (SOP) establishes comprehensive protocols for managing Safety Data Sheets (SDS) to ensure compliance with OSHA's Hazard Communication Standard (29 CFR 1910.1200) and protect Florida First Roofing LLC employees from chemical hazards in roofing operations.

### 1.2 Scope
This SOP applies to all Florida First Roofing LLC operations involving:
- Chemical storage and handling
- SDS collection and maintenance
- Employee access to hazard information
- Digital and physical SDS management systems
- Training and communication requirements
- Florida Department of Health chemical reporting

## 2. DEFINITIONS

- **SDS (Safety Data Sheet)**: Standardized document providing essential health and safety information about chemical products
- **HazCom**: OSHA Hazard Communication Standard
- **Chemical Inventory**: Complete listing of all hazardous chemicals used in operations
- **Digital SDS Management**: Electronic system for storing and accessing safety data sheets
- **GHS**: Globally Harmonized System of Classification and Labelling of Chemicals

## 3. RESPONSIBILITIES

### 3.1 Safety Manager
- Maintain comprehensive SDS library
- Ensure current versions of all SDS documents
- Coordinate with suppliers for SDS updates
- Train employees on SDS interpretation
- Conduct annual SDS system audits

### 3.2 Project Managers
- Verify SDS availability for all project chemicals
- Ensure field teams have access to relevant SDS
- Report new chemical products to Safety Manager
- Conduct pre-job SDS reviews

### 3.3 Field Supervisors
- Maintain job site SDS accessibility
- Review SDS with crew before chemical use
- Report chemical exposures or incidents
- Ensure proper chemical storage per SDS requirements

### 3.4 All Employees
- Understand SDS information for chemicals they use
- Report missing or outdated SDS documents
- Follow safety precautions outlined in SDS
- Participate in SDS training programs

## 5. PROCEDURES

### 5.1 SDS Collection and Acquisition

#### 5.1.1 Initial Collection
1. **Chemical Inventory Creation**
   - Catalog all chemicals used in operations
   - Include brand names, generic names, and chemical compositions
   - Document suppliers and manufacturers
   - Assign unique identification numbers

2. **SDS Request Process**
   - Contact manufacturers/suppliers for current SDS
   - Use standardized SDS request template
   - Specify Florida First Roofing LLC requirements
   - Set 30-day response deadline

3. **Verification Procedures**
   - Confirm SDS accuracy and completeness
   - Verify 16-section GHS format compliance
   - Check revision dates and version numbers
   - Validate chemical identity matching

#### 5.1.2 Digital Management System
1. **Database Setup**
   - Implement SDS management software
   - Create searchable chemical database
   - Establish user access controls
   - Configure automatic update alerts

2. **Document Processing**
   - Scan physical SDS documents
   - Apply optical character recognition (OCR)
   - Create searchable PDF files
   - Implement version control system

3. **Mobile Access Configuration**
   - Deploy mobile SDS applications
   - Sync with central database
   - Enable offline access capability
   - Implement GPS-based SDS alerts

### 5.2 SDS Maintenance and Updates

#### 5.2.1 Regular Review Schedule
1. **Monthly Reviews**
   - Check for supplier SDS updates
   - Verify new product additions
   - Update chemical inventory
   - Review access logs and usage

2. **Quarterly Audits**
   - Comprehensive SDS library review
   - Verify all chemicals have current SDS
   - Check digital system functionality
   - Update training materials

3. **Annual Assessments**
   - Complete chemical inventory reconciliation
   - Evaluate SDS management effectiveness
   - Update procedures based on regulatory changes
   - Conduct supplier performance review

#### 5.2.2 Update Procedures
1. **Automatic Updates**
   - Configure supplier notification systems
   - Implement automated SDS retrieval
   - Set up version change alerts
   - Archive superseded versions

2. **Manual Verification**
   - Review automated updates for accuracy
   - Verify chemical composition changes
   - Update hazard classifications
   - Communicate significant changes to staff

### 5.3 Employee Access and Training

#### 5.3.1 Access Procedures
1. **Physical Access**
   - Maintain SDS binders at all work locations
   - Ensure 24/7 accessibility during operations
   - Provide weather-resistant storage
   - Include emergency contact information

2. **Digital Access**
   - Provide login credentials to all employees
   - Ensure mobile device compatibility
   - Configure offline access for field work
   - Maintain backup access methods

#### 5.3.2 Training Programs
1. **Initial Training**
   - SDS format and section explanations
   - Hazard identification and classification
   - Emergency response procedures
   - Personal protective equipment selection

2. **Ongoing Education**
   - Annual refresher training
   - New chemical orientation
   - Incident-based training updates
   - Regulatory change notifications

### 5.4 Emergency Procedures

#### 5.4.1 Chemical Exposure Response
1. **Immediate Actions**
   - Consult relevant SDS Section 4 (First Aid)
   - Implement emergency response procedures
   - Contact emergency medical services if required
   - Notify Safety Manager and management

2. **Documentation Requirements**
   - Complete exposure incident report
   - Attach relevant SDS to report
   - Document medical treatment provided
   - Report to OSHA if required

#### 5.4.2 Spill Response
1. **SDS-Based Response**
   - Consult SDS Section 6 (Accidental Release)
   - Implement containment procedures
   - Use specified cleanup materials
   - Follow disposal requirements

2. **Notification Procedures**
   - Contact local emergency responders if required
   - Notify Florida Department of Environmental Protection
   - Report to EPA if above reportable quantities
   - Document response actions taken

## 6. DOCUMENTATION AND RECORD KEEPING

### 6.1 Required Records
- Complete SDS library (physical and digital)
- Chemical inventory with SDS cross-references
- Employee training records
- SDS access logs and usage tracking
- Update and revision history
- Incident reports involving chemicals

### 6.2 Record Retention
- Current SDS: Maintain as long as chemical is used
- Superseded SDS: Retain for 30 years
- Training records: Retain for duration of employment plus 30 years
- Incident reports: Permanent retention
- Inventory records: 30 years
- Access logs: 5 years

### 6.3 Record Location
- Central office: Master SDS library and database
- Project sites: Relevant SDS for chemicals on-site
- Mobile devices: Digital access to complete library
- Emergency services: Emergency contact SDS list
- Regulatory agencies: As required by law
