## BLOCK 3: DEFINITIONS - Key Terms and Concepts

**Notice of Acceptance (NOA):** Miami-Dade County approval certifying that Metal Tile Profile System has passed stringent testing and is approved for HVHZ installations per NOA 22-1201.04.

**High Velocity Hurricane Zone (HVHZ):** Geographic areas subject to wind speeds of 120 mph or greater, requiring enhanced installation standards and NOA-approved materials.

**Snap-Lock Mechanical Seaming:** Mechanical snap-lock seaming system providing weathertight panel connections with thermal expansion capability.

**HVHZ Installation Pattern:** Enhanced 6-nail installation pattern required for wind speeds up to 140 mph, exceeding standard 4-nail residential requirements.

**Wind Load Resistance:** System capability to withstand design wind pressures without failure, verified through ASTM D3161 and D7158 testing protocols.

**Solid Wood Deck Requirement:** Minimum 15/32" OSB or plywood sheathing required for proper fastener engagement and wind uplift resistance.

## BLOCK 5: PROCEDURE - Step-by-Step Implementation Guide

### 5.1 Pre-Installation Preparation and NOA Verification

**5.1.1 Documentation Requirements:**
1. Verify NOA 22-1201.04 is current and valid through undefined
2. Confirm Tri County Metals Metal Tile Profile System shipment matches NOA specifications
3. Verify Miami-Dade Product Control Approved labeling on all materials
4. Obtain complete manufacturer installation instructions
5. Prepare NOA documentation for on-site Building Official access

**5.1.2 Deck and Substrate Preparation:**
1. Inspect deck for solid wood construction minimum 15/32"
2. Verify deck meets or exceeds HVHZ structural requirements
3. Confirm proper deck fastening and structural integrity
4. Install approved underlayment system per manufacturer specifications
5. Complete ventilation system design for HVHZ requirements

**5.1.3 Material Verification and Setup:**
1. Verify Metal Tile Profile panels display Miami-Dade approval labeling
2. Check manufacturer date codes and proper storage conditions (protect from moisture)
3. Confirm attachment specifications: structural clips and concealed fasteners per manufacturer
4. Set up snap-lock seaming tools and panel handling equipment
5. Prepare HVHZ-specific metal attachment clips and fastening systems

### 5.2 Snap-Lock Mechanical Seaming Installation Procedures

**5.2.1 Eave Edge and Drip Edge Installation:**
1. Install metal eave trim and drip edge components per manufacturer specifications
2. Apply Snap-Lock Mechanical Seaming preparation procedures for panel attachment
3. Install eave closure strips with proper ventilation allowances
4. Position structural clips at maximum 12" centers per HVHZ requirements
5. Verify proper alignment for first metal tile profile panel installation

**5.2.2 Metal Tile Profile Panel Installation with HVHZ Attachment:**
1. Install structural attachment clips at 12" maximum spacing per HVHZ wind load requirements
2. Position 16" metal tile profile panels with proper thermal expansion joints (1/4" minimum)
3. Engage snap-lock seaming system ensuring complete mechanical connection
4. Verify 1.5" seam height for weather-tight panel connections
5. Install concealed clip fastening system to prevent thermal bridging and water penetration
6. Complete panel edge securement with HVHZ-approved metal termination details

**5.2.3 Metal Ridge and Hip Component Installation:**
1. Install metal ridge caps and hip components with HVHZ-rated wind resistance
2. Apply Snap-Lock Mechanical Seaming to all ridge panel connections
3. Use manufacturer-specified concealed clip fasteners for ridge attachment
4. Maintain proper metal component overlap (minimum 6") and thermal expansion allowances
5. Verify wind uplift ratings for all metal ridge system components per ASTM E1592

### 5.3 Quality Control and Final Verification

**5.3.1 Installation Quality Verification:**
1. Inspect Snap-Lock Mechanical Seaming implementation throughout roof
2. Verify 6-nail pattern compliance on random sample
3. Confirm all fasteners are properly driven (flush, not overdriven)
4. Check manufacturer-specified exposure rates and overlap requirements
5. Verify no staples were used (prohibited in HVHZ installations)

**5.3.2 NOA Compliance Documentation:**
1. Complete NOA 22-1201.04 compliance checklist
2. Document Snap-Lock Mechanical Seaming implementation
3. Photograph key installation details for compliance records
4. Verify Miami-Dade labeling requirements are met
5. Prepare documentation package for Building Official inspection
