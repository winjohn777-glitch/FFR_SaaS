## BLOCK 3: DEFINITIONS - Key Terms and Concepts

**Notice of Acceptance (NOA):** Miami-Dade County approval certifying that Standing Seam Aluminum Roof System has passed stringent testing and is approved for HVHZ installations per NOA 23-0915.01.

**High Velocity Hurricane Zone (HVHZ):** Geographic areas subject to wind speeds of 120 mph or greater, requiring enhanced installation standards and NOA-approved materials.

**Concealed Clip Fastening System:** Concealed clip fastening system with thermal movement accommodation for superior metal panel attachment.

**HVHZ Clip Attachment Pattern:** Enhanced concealed clip attachment system at 12" maximum spacing required for wind speeds up to 175 mph (Vasd) / 220 mph (Vult), exceeding standard commercial requirements.

**Wind Load Resistance:** System capability to withstand design wind pressures without failure, verified through ASTM E1592 and UL 580 uplift testing protocols.

**Solid Wood Deck Requirement:** Minimum Structural deck minimum 15/32" plywood/OSB sheathing required for proper fastener engagement and wind uplift resistance.

## BLOCK 5: PROCEDURE - Step-by-Step Implementation Guide

### 5.1 Pre-Installation Preparation and NOA Verification

**5.1.1 Documentation Requirements:**
1. Verify NOA 23-0915.01 is current and valid through 2028-09-15
2. Confirm Gulf Coast Supply Standing Seam Aluminum Roof System shipment matches NOA specifications
3. Verify Miami-Dade Product Control Approved labeling on all materials
4. Obtain complete manufacturer installation instructions
5. Prepare NOA documentation for on-site Building Official access

**5.1.2 Deck and Substrate Preparation:**
1. Inspect deck for Structural deck minimum 15/32" plywood/OSB
2. Verify deck meets or exceeds HVHZ structural requirements
3. Confirm proper deck fastening and structural integrity
4. Install approved underlayment system per manufacturer specifications
5. Complete ventilation system design for HVHZ requirements

**5.1.3 Material Verification and Setup:**
1. Verify Standing Seam Aluminum Roof System bundles display Miami-Dade approval labeling
2. Check manufacturer date codes and storage condition requirements
3. Confirm clip specifications: structural seaming clips with thermal expansion capability
4. Set up standing seam installation tools including seaming machines and clip positioning equipment
5. Prepare HVHZ-specific installation equipment and thermal expansion gauges

### 5.2 Concealed Clip Fastening System Installation Procedures

**5.2.1 Eave Edge and Base Flashing Installation:**
1. Install eave edge trim and drip edge components per manufacturer specifications
2. Apply base flashing with proper thermal expansion joints (minimum 1/4")
3. Position eave closure strips with adequate ventilation allowances
4. Install structural attachment clips at maximum 12" centers per HVHZ requirements
5. Verify proper alignment for first standing seam panel installation

**5.2.2 Standing Seam Panel Installation with HVHZ Attachment:**
1. Install structural seaming clips at 12" maximum spacing per HVHZ wind load requirements
2. Position aluminum panels with proper thermal expansion joints (1/4" minimum)
3. Engage mechanical seaming system ensuring complete weather-tight connections
4. Verify seam height requirements (typically 1.5" minimum) for wind uplift resistance
5. Install concealed clip system to allow thermal movement while maintaining attachment strength
6. Complete panel edge securement with HVHZ-approved metal termination details

**5.2.3 Ridge Cap and Hip Components Installation:**
1. Install aluminum ridge caps and hip components with HVHZ-rated wind resistance
2. Apply mechanical seaming to all ridge panel connections with thermal expansion capability
3. Use manufacturer-specified concealed clip fasteners for ridge attachment
4. Maintain proper metal component overlap (minimum 6") and thermal expansion allowances
5. Verify wind uplift ratings for all ridge system components per ASTM E1592

### 5.3 Quality Control and Final Verification

**5.3.1 Installation Quality Verification:**
1. Inspect Concealed Clip Fastening System implementation throughout roof
2. Verify concealed clip attachment compliance on random sample
3. Confirm all fasteners are properly driven (flush, not overdriven)
4. Check manufacturer-specified exposure rates and overlap requirements
5. Verify no staples were used (prohibited in HVHZ installations)

**5.3.2 NOA Compliance Documentation:**
1. Complete NOA 23-0915.01 compliance checklist
2. Document Concealed Clip Fastening System implementation
3. Photograph key installation details for compliance records
4. Verify Miami-Dade labeling requirements are met
5. Prepare documentation package for Building Official inspection
