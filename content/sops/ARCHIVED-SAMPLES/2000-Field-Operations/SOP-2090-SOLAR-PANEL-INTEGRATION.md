# SOP-2090: Solar Panel Integration Procedures

## 1. PURPOSE AND SCOPE

### 1.1 Purpose
This Standard Operating Procedure (SOP) establishes comprehensive protocols for integrating solar photovoltaic (PV) systems with roofing installations to ensure structural integrity, weather protection, and optimal energy production. This procedure addresses both new construction and retrofit applications.

### 1.2 Scope
This SOP applies to all Florida First Roofing LLC projects involving:
- Solar panel mounting system installation
- Roof penetration sealing and flashing
- Structural load assessment and reinforcement
- Electrical safety and coordination
- Building-integrated photovoltaic (BIPV) systems
- Solar thermal system integration

## 2. DEFINITIONS

- **Photovoltaic (PV)**: Technology that converts sunlight directly into electricity
- **Building-Integrated Photovoltaics (BIPV)**: Solar panels that serve as building materials
- **Mounting System**: Hardware that secures solar panels to the roof structure
- **Microinverter**: Device that converts DC power from individual solar panels to AC power
- **String Inverter**: Device that converts DC power from multiple connected solar panels
- **Net Metering**: System allowing excess solar power to be fed back into the electrical grid

## 5. PROCEDURES

### 5.1 Pre-Installation Assessment

#### 5.1.1 Structural Analysis
1. **Load Calculation**
   - Calculate dead load from solar panels and mounting system
   - Assess wind uplift forces on solar array
   - Evaluate snow load requirements (where applicable)
   - Determine seismic load considerations

2. **Roof Structure Evaluation**
   - Inspect rafters, trusses, and sheathing condition
   - Verify adequate structural capacity for additional loads
   - Assess need for structural reinforcement
   - Determine optimal mounting point locations

3. **Roof Condition Assessment**
   - Evaluate roof age and remaining useful life
   - Identify any needed repairs before solar installation
   - Assess compatibility with solar mounting systems
   - Plan for future roof maintenance access

#### 5.1.2 Design Coordination
1. **Solar Array Layout**
   - Optimize panel placement for maximum production
   - Consider shading from HVAC equipment and other obstructions
   - Plan electrical routing and conduit pathways
   - Maintain required setbacks and access ways

2. **Mounting System Selection**
   - Choose appropriate mounting system for roof type
   - Verify structural attachment capacity
   - Ensure compatibility with roofing materials
   - Plan for thermal expansion and movement

### 5.2 Mounting System Installation

#### 5.2.1 Attachment Point Preparation
1. **Location Marking**
   - Use structural drawings to locate rafters/trusses
   - Mark mounting points with precision
   - Verify clearances from roof edges and penetrations
   - Check for utilities or obstructions in attachment areas

2. **Pilot Hole Drilling**
   - Use appropriate drill bits for fastener type
   - Drill pilot holes to prevent wood splitting
   - Verify adequate fastener penetration depth
   - Clean debris from holes before fastener installation

#### 5.2.2 Flashing and Sealing Procedures
1. **Flashing Installation**
   - Install flashing under each mounting point
   - Use compatible materials (galvanized steel, aluminum, or stainless steel)
   - Ensure proper overlap with existing roofing materials
   - Follow manufacturer specifications for flashing details

2. **Sealant Application**
   - Apply appropriate sealant around mounting hardware
   - Use UV-resistant, paintable sealants
   - Ensure complete coverage of penetration area
   - Allow proper curing time before panel installation

3. **Quality Control**
   - Inspect each mounting point for proper sealing
   - Test flashing integration with water flow
   - Verify fastener torque specifications
   - Document installation with photographs

#### 5.2.3 Grounding and Bonding
1. **Equipment Grounding**
   - Install grounding lugs on each mounting rail
   - Connect all metallic components to grounding system
   - Use listed grounding hardware and connectors
   - Verify continuity throughout grounding system

2. **System Bonding**
   - Bond all metal components of mounting system
   - Install grounding electrode conductor connections
   - Connect to building's electrical grounding system
   - Test and document grounding resistance

### 5.3 Panel Installation and Integration

#### 5.3.1 Panel Mounting Procedures
1. **Panel Handling**
   - Use appropriate lifting and handling equipment
   - Protect panels from damage during installation
   - Verify panel orientation and spacing
   - Check for damage before installation

2. **Attachment Methods**
   - Secure panels to mounting rails with appropriate clamps
   - Maintain required spacing between panels
   - Ensure proper drainage channels
   - Allow for thermal expansion

#### 5.3.2 Electrical Integration
1. **DC Wiring**
   - Route wiring through protective conduit
   - Maintain proper bend radius for cables
   - Use weatherproof connectors and junction boxes
   - Label all circuits according to NEC requirements

2. **Inverter Installation**
   - Mount inverters in accessible, well-ventilated locations
   - Provide adequate clearances for maintenance
   - Install disconnect switches as required
   - Connect to building electrical system through proper breakers

### 5.4 Building-Integrated Photovoltaics (BIPV)

#### 5.4.1 BIPV Installation
1. **Structural Integration**
   - Install BIPV modules as integral building components
   - Ensure proper structural support and attachment
   - Maintain building envelope continuity
   - Provide adequate ventilation behind modules

2. **Weather Sealing**
   - Install weatherproofing systems between modules
   - Use appropriate gaskets and sealants
   - Test for water resistance under simulated conditions
   - Provide thermal expansion accommodation

#### 5.4.2 Aesthetic Integration
1. **Visual Continuity**
   - Align modules with building architectural features
   - Maintain consistent module spacing and orientation
   - Coordinate with other building materials and colors
   - Minimize visible mounting hardware and wiring

### 5.5 Quality Control and Testing

#### 5.5.1 Installation Inspection
1. **Structural Verification**
   - Verify all mounting points are properly secured
   - Check for adequate fastener penetration
   - Inspect flashing and sealing integrity
   - Test mounting system stability

2. **Electrical Testing**
   - Test system grounding and bonding
   - Verify proper circuit connections
   - Check polarity and voltage readings
   - Test rapid shutdown functionality

#### 5.5.2 Performance Testing
1. **Initial System Testing**
   - Measure open-circuit voltage and short-circuit current
   - Test inverter operation and monitoring systems
   - Verify net meter installation and operation
   - Document baseline performance measurements

2. **Integration Testing**
   - Test interaction between roofing and solar systems
   - Verify weather resistance under water testing
   - Check thermal performance and ventilation
   - Monitor for any interference or operational issues
